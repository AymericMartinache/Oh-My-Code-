Questions Fréquentes

Et si je n'ai qu'un seuk ékélent, est-ce que j'obtiens malgré tout un tableau ?

Oui ! Avec "querySelectorAll" on obtient un tableau même s'il ne contient qu'un seul élément.

Et si j'ajoute d'autres questions ?

Le code JavaScript est conçu pour s'adapter. C'est la notion de maintenabilité. Le nombre de questions peut changer, mais le code fonctionne de la même manière. Attention ! Il faut garder la structure html pour les questions / réponses.

Je voudrais affecter le même programme à plusieurs éléments, mais ils n'ont pas la m^me classe, comment faire ?

Ajouter une classe ! Il est possible d'ajouter une classe dans le seul but de créer un programme, même si cela ne correspond à aucun style CSS.
